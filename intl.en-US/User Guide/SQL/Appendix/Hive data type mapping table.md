# Hive data type mapping table {#concept_opw_2vf_vdb .concept}

The data type mapping table for MaxCompute and hive is as follows:

|Hive Data Type|MaxCompute Data Type|
|:-------------|:-------------------|
|BOOLEAN|Boolean|
|TINYINT|Tinyint|
|SMALLINT|Smallint|
|INT|Int|
|BIGINT|Bigint|
|FLOAT|Float|
|DOUBLE|Double|
|Decimal|Decimal|
|String|String|
|Varchar|Varchar|
|Char|String|
|BINARY|Binary|
|Timestamp|Timestamp|
|Date|Datetime|
|ARRAY|Array|
|`Map <key, value>`|MAP|
|STRUCT|STRUCT|
|Union|This feature is not supported.|
